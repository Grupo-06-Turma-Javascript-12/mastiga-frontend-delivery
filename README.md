## 📌 Sobre o Projeto

O **Mastiga** é uma aplicação web de delivery desenvolvida como projeto de conclusão de módulo. A plataforma permite que usuários naveguem por um cardápio, adicionem itens ao carrinho e realizem pedidos de forma intuitiva.

O projeto foi construído com foco em boas práticas de desenvolvimento frontend, componentização, tipagem estática com TypeScript e deploy contínuo via Vercel.

---

## ✨ Funcionalidades

- 🏠 Página inicial com vitrine de produtos
- 🛒 Carrinho de compras interativo
- 📋 Cardápio com categorias e itens
- 📱 Layout responsivo para mobile e desktop
- ⚡ Navegação rápida com roteamento client-side

## 🚀 Como Rodar Localmente

### Pré-requisitos

- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

### Passo a passo

```bash
# 1. Clone o repositório
git clone https://github.com/Grupo-06-Turma-Javascript-12/mastiga-frontend-delivery.git

# 2. Acesse a pasta do projeto
cd mastiga-frontend-delivery

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`.

### Scripts disponíveis

```bash
npm run dev      # Inicia o servidor de desenvolvimento
npm run build    # Gera o build de produção
npm run preview  # Visualiza o build localmente
npm run lint     # Executa o ESLint
```
